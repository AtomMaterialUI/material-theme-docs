## Additional Plugins

The Material Theme UI has two subscription models:
- **Free** - The plugin is available for free but with limited features
- **Premium** - All features are available.

However, there is another subscription model available that allows you to use the free model but purchase premium features separately, through the use of **Material Theme UI Plugins**.

These plugins are the following:
- [Material Theme High Contrast](https://plugins.jetbrains.com/plugin/17456-material-theme-ui-high-contrast): Unlock the **High Contrast** feature.
- [Material Theme Extras](https://plugins.jetbrains.com/plugin/19250-material-theme-ui-extras): Unlock the **Tabs**, **Trees**, **UI Components**, **Spacing**, and a few other premium customization features.
- [Material Theme Project Frame](https://plugins.jetbrains.com/plugin/19310-material-theme-ui-project-frame): Unlock the **Project Frame** customization features.
- [Material Theme Custom Theme](https://plugins.jetbrains.com/plugin/19308-material-theme-ui-custom-theme): Unlock the **Custom Theme** and **Light Custom Theme** features.
- [Material Theme Language Additions](https://plugins.jetbrains.com/plugin/19309-material-theme-ui-language-additions): Unlock the **Language Additions** feature.
- [Material Theme Randomizer](https://plugins.jetbrains.com/plugin/28143-material-theme-ui-theme-randomizer): Unlock the **Theme Randomizer** feature.
- [Material Theme Islands](https://plugins.jetbrains.com/plugin/28142-material-theme-ui-islands): Unlock Islands customization features.

When you purchase a license of one of these plugins while having the Free Version, you'll get access to the features in question without having to purchase a Premium License! 

You can then find the plugins you purchased in this page.

{% include figure.html content="/screens/home.png" caption="License Information" %}

**Note**: If you purchase an additional plugin while being on the Premium Version, it will simply do nothing as all features are already available in the Premium Version. Be careful!
