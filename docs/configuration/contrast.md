Contrast settings allow you to improve the visibility of UI components by adding more contrast to specific panels and elements.
{:class='title'}

{% include carbonads.html %}

## Contrast Settings

You can configure the contrast settings by navigating to `Settings → Appearance → Material Theme UI → Contrast`.

{% include figure.html content="/screens/settingsv3/contrastMode.png" caption="Contrast Settings" %}

### Contrast Mode

When **Contrast Mode** is enabled, the plugin applies a more contrasted color to specific components and panels, such as the Project View, Tool Windows, and Tabbed Panes.

This helps in distinguishing different areas of the IDE more easily, especially in dark themes.

{% include figure.html content="/screens/contrast/withContrast.png" caption="With Contrast" %}

{% include figure.html content="/screens/contrast/noContrast.png" caption="No Contrast" %}

### High Contrast

This feature is only available for premium users.
{:class='card-panel warn'}

**High Contrast** takes it a step further by increasing the contrast even more. It makes some colors brighter (or darker for light themes) to ensure that text and icons are as prominent as possible.

### Contrast Value

This feature is only available for premium users.
{:class='card-panel warn'}

The **Contrast Value** slider allows you to fine-tune the level of contrast applied when *High Contrast* is enabled.

- **Range**: 0 to 8
- **Effect**: Higher values result in a more pronounced contrast between different UI elements.
