## What's new in Material Theme UI v10.10.0

This version includes a quick hotfix to fix some urgent issues regarding the screenshots and the tool window opacity. But we've also delivered small improvements for the Rounded Tabs.

### Rounded Tabs Padding and Radius

This version includes two new settings to control the padding and the radius of the rounded tabs.

We've changed the domain name for the screenshots, in order to serve them more efficiently. We've also fixed the issue with the tool windows opacity on Islands mode that was not being applied correctly
